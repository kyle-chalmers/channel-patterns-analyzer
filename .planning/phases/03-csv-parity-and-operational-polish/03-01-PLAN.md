## Phase Goal (User Story)

**As a** solo channel operator without BigQuery auth set up locally, **I want to** run the analyzer in CSV-mode and get a structurally identical weekly report drawn from `sample_data/*.csv`, **so that** I can develop and demo the analyzer without burning real BigQuery quota or leaking real channel data into screenshots.

This plan is one of three feeding that user story; the demo-runnable behavior lands when Plan 03-03 wires CSV into the recipe.
