## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| operator → /run-analyzer | Operator passes no arguments; `$ARGUMENTS` is ignored per Step 0 |
| recipe → BigQuery (bq CLI or MCP) | Authenticated transport; recipe substitutes only `$BQ_DATASET` into SQL files, never operator input |
| recipe → write-notion-report Skill | Recipe builds a strict 8-key dict; Skill validates before any Notion call |
| recipe → file system (reports/, runs/) | Writes inside the repo root only; no path traversal possible since paths are formed from `run_date` (date string) |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-01-03-01 | Tampering | SQL dataset substitution | mitigate | Recipe substitutes ONLY the literal `youtube_analytics` token with `$BQ_DATASET` value (read from `.env`); never substitutes operator-supplied text into SQL |
